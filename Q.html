开发备忘录 · 从原型到上线

  • 后端聚合: 每日凌晨ETL汇总订单/财务库，生成运营日报表 (Python/Node.js + SQL)
  • API设计: RESTful 接口 /api/daily/dashboard 返回KPI、趋势、流水 (JWT 鉴权)
  • 前端架构: 可选用 Vue3 或 React 重构当前原型，封装图表组件 (Chart.js 或 ECharts)
  • 权限与审计: 老板只读账号，敏感财务字段脱敏? 实际生产环境可记录访问日志
  • 增强指标: 后续可加环比/同比、成本构成、现金流预测，支持日期切换
  • 告警订阅: 当净利润或现金流波动超阈值时，自动发送企业微信提醒
  • 响应式优化: 目前适配Pad/桌面，移动端进一步调整字体与布局
当前静态原型演示数据 (硬编码) — 实际接入API后动态渲染 每日数据刷新: 后端缓存至Redis，减少数据库压力
