    🎁 个人交易奖励（比例制）

    每月10日、20日、30日根据累计交易量按比例发放H代币：

    • ≥ 10,000 USDT → 奖励总额的 0.5% 等值H
    • ≥ 30,000 USDT → 奖励总额的 0.8% 等值H
    • ≥ 50,000 USDT → 奖励总额的 1% 等值H
    • ≥ 100,000 USDT → 奖励总额的 1.2% 等值H

    示例：累计交易40,000 USDT，奖励 = 40,000 × 0.8% = 320 H

    👥 团队奖励规则

    加权团队交易量 = 本人100% + 一级50% + 二级10%

    • ≥ 50,000 USDT → 奖励加权总量的 0.5%
    • ≥ 100,000 USDT → 奖励加权总量的 1%

    累计制，每月10/20/30日随个人奖励一并发放。
